Welcome to my salary prediction webapp

Enter years of working experience of employee

About application

Application is based on linear regression model of Machine Learning, used to predict salary of employee based on his/her working experience(in years).

Technology used

Flask:- Flask framework for web app development.
Machine Learning:- To create machine Learning model.
Container Techonology:- I used Docker Container engine to create Container and deploy my web application.
Cloud Computing:- I used cloud platform(AWS) to run my docker container.
Kubernetes:- I used kubernetes to launch/expose/manage my docker container.
Git & Github:- I push/upload my code in github repository. And used repository url to download my application from github to docker container.
